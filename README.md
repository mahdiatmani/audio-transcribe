# TranscribeAI - Audio Transcription SaaS

AI-powered audio transcription platform with subscription plans.

## Features
- 99+ language support
- Free & paid subscription tiers
- Stripe payment integration
- Usage tracking & limits

## Tech Stack
- **Backend**: Flask, Whisper AI, Stripe
- **Frontend**: React, Tailwind CSS
- **Database**: PostgreSQL/SQLite

## Quick Start

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```

Visit http://localhost:3000
